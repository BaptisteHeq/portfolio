SAE 3.01 - Génération de diagrammes de classes Java

En groupe de 4, nous avons conçu une application JavaFX capable d’analyser dynamiquement les classes d’un projet Java (via l’introspection) pour en générer automatiquement un diagramme de classes clair et modifiable. L'application permet de visualiser les relations (héritage, association), de réorganiser les éléments via une interface graphique, et d’exporter le tout en image ou en PlantUML. Le projet incluait une phase d’analyse, de planification sur Trello et une gestion en itérations.
