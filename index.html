SAE 3.01 - Génération de diagrammes de classes Java

En groupe de 4, nous avons conçu une application JavaFX capable d’analyser dynamiquement les classes d’un projet Java (via l’introspection) pour en générer automatiquement un diagramme de classes clair et modifiable. L'application permet de visualiser les relations (héritage, association), de réorganiser les éléments via une interface graphique, et d’exporter le tout en image ou en PlantUML. Semestre 4

SAE 4.01 - Projet IA Détection de biomes sur des exoplanètes

Dans cette SAE, nous avons développé une application permettant de détecter automatiquement les biomes présents à la surface d’exoplanètes à partir d’images issues d’un télescope gravitationnel simulé. L’objectif était de :

  • Prétraiter les images (floutage par convolution ou filtre gaussien) pour lisser les zones de couleur,
  • Appliquer des algorithmes de clustering afin d’identifier les zones de couleur homogène représentant les différents biomes (forêts, déserts, lacs, etc.),
  • Étiqueter ces zones selon une correspondance couleur/biome,
  • Détecter, pour chaque biome, les écosystèmes indépendants spatialement via un second clustering sur les coordonnées des pixels,
  • Visualiser dynamiquement les résultats avec des cartes colorées et lisibles.

Le tout était encapsulé dans une application interactive permettant d’analyser de nouvelles cartes planétaires. Ce projet mêlait traitement d’image, classification non supervisée (clustering) et visualisation de données, avec une attention particulière portée à l’ergonomie de l’outil pour un usage scientifique.
